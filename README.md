# Office-Expense-Dashboard
The Office Expense Tracker is a professional Excel-based analytical tool designed to streamline financial oversight by transforming raw transaction logs into an interactive visual dashboard. By leveraging a structured data architecture, the project provides a granular look at departmental spending and monthly budget trends to help identify cost-saving opportunities.
